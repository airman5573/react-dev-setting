# React Dev Setting

## 동기

1. Create-React-App없이 개발 환경을 구성함으로써 웹 개발의 플로우를 이해합니다
2. 딱 필요한 설정들만 취해서 프로젝트의 사이즈를 줄이고 속도를 높입니다
3. CRA에 의존을 하지 않음으로써, 미래에 발생할 에러에 더 능숙하게 대처할 수 있습니다
